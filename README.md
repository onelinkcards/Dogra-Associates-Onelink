# Dogra Associates — OneLink card

## Repository

```bash
git clone https://github.com/onelinkcards/Dogra-Associates-Onelink.git
cd Dogra-Associates-Onelink
pnpm install
pnpm dev
```

## Scripts

| Command     | Description        |
| ----------- | ------------------ |
| `pnpm dev`  | Dev server (3000)  |
| `pnpm build`| Production build   |
| `pnpm start`| Start production   |
| `pnpm lint` | ESLint             |
